## 🌟 About ASK-VISA

**ASK-VISA** is an advanced, automated immigration and visa consultation system designed to simplify and streamline the application process. Utilizing intelligent algorithms, the platform guides applicants through their specific requirements, manages document processing, supports automated invoice and PDF generation, and securely processes payments via Razorpay integration.

</div>

<br/>

## 🚀 Key Features

*   🤖 **AI-Driven Guidance:** Smart questionnaires to determine exact visa requirements for supported countries.
*   📄 **Automated Document Processing:** Real-time PDF generation for visa applications, forms, and invoices using TCPDF.
*   💳 **Secure Payments:** Integrated Razorpay checkout for seamless transaction processing and robust security.
*   📧 **Automated Notifications:** Email alerts, applicant tracking, and updates using PHPMailer to keep applicants informed.
*   📊 **Admin Dashboard:** Comprehensive control panel for reviewing applications, generating reports, and managing applicants seamlessly.

## ⚙️ Tech Stack & Architecture

- **Backend Logic:** PHP (Native architecture)
- **Database:** MySQL
- **Frontend Experience:** HTML5, CSS3, JavaScript (ES6)
- **Document Rendering:** TCPDF (Dynamic PDF Generation)
- **Email Infrastructure:** PHPMailer (Automated Communication)
- **Payments:** Razorpay API Interface
- **E2E Testing:** Puppeteer, Node.js

<br/>

## 🛠️ Quick Start & Installation

To run the application locally on your Apache/Nginx environment, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/walterhydra/ASK-VISA.git
   ```

2. **Database Configuration:**
   - Ensure your MySQL service is running.
   - Run the provided `public_html/local_db.sqlite` script or manual `.sql` files (`run_sql_local.php`).
   - Open `public_html/config.php` and `public_html/db.php` and update your MySQL connection credentials.

3. **Install Testing Dependencies (Optional):**
   *(If you wish to run the Puppeteer automation tests)*
   ```bash
   npm install
   ```

4. **Launch the Service:**
   - Map your local web server (XAMPP, WAMP, Laragon, etc.) directory to point to the clone directory.
   - Open your browser to `http://localhost/YOUR_FOLDER/public_html`.
